# invoice+ AltStore 配布リポジトリ

電子帳簿保存法準拠の請求書管理モバイルアプリ「**invoice+**」のAltStore配布用リポジトリです。

## 📱 アプリについて

invoice+ は、社内LANで動作する請求書管理システムのモバイルクライアントアプリです。

### 主な機能

- 📸 **書類スキャン**: カメラを使用した高品質スキャン
- 📏 **自動サイズ判定**: A4/A5/レシート等の自動識別
- 🔍 **解像度チェック**: 電帳法要件（200dpi以上）の自動検証
- 🔒 **セキュア通信**: JWT認証とHTTPS通信
- ⚙️ **サーバー設定**: アプリ内で接続先を動的に変更可能

## 🚀 インストール方法

### 1. AltStore のインストール

まず、デバイスに [AltStore](https://altstore.io/) をインストールしてください。

### 2. ソースの追加

AltStore アプリを開き、以下の手順でソースを追加します:

1. **Browse** タブを開く
2. 右上の **+** アイコンをタップ
3. 以下のURLを入力:
   ```
   https://raw.githubusercontent.com/<ORG>/invoice-plus-altstore/main/apps.json
   ```

### 3. アプリのインストール

1. ソース一覧から **TigerRack Apps** を選択
2. **invoice+** を見つけて **GET** をタップ
3. インストール完了を待つ

## ⚙️ 初回設定

アプリをインストール後、初回起動時に以下の設定を行ってください:

1. ログイン画面の右上にある **⚙️ 設定アイコン** をタップ
2. **Base URL** に以下を入力:
   ```
   https://192.168.24.151:9005/api
   ```
3. **保存して反映** をタップ
4. 社内ネットワーク(Wi-Fi)に接続していることを確認
5. 従業員コードとパスワードでログイン

## 🔄 アプリの更新

AltStore は7日ごとにアプリの再署名が必要です。

### 自動更新 (推奨)

- 同一ネットワーク上で **AltServer** (PC/Mac) を起動しておく
- バックグラウンドで自動的に更新されます

### 手動更新

1. AltStore アプリを開く
2. **My Apps** タブを開く
3. **Refresh All** をタップ

## 📋 システム要件

- **iOS**: 15.0 以上
- **ネットワーク**: 社内LAN (Wi-Fi) 必須
- **サーバー**: `192.168.24.151:9005` にアクセス可能であること

## 🔧 トラブルシューティング

### ログインできない

1. 設定アイコンから接続先URLを確認
2. 社内Wi-Fiに接続しているか確認
3. サーバーが稼働しているか確認

### アプリが起動しない

1. AltStore で **Refresh** を実行
2. デバイスを再起動
3. アプリを再インストール

### 7日後に起動できなくなった

- AltServer が起動しているPC/Macと同じネットワークに接続
- AltStore で **Refresh All** を実行
